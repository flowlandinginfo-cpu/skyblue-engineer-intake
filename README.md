สำหรับ Sky Blue Construction

## Quick start

```bash
# 1. Install dependencies
npm install

# 2. Copy env file and fill in Supabase anon key
cp .env.local.example .env.local
# Edit .env.local — paste NEXT_PUBLIC_SUPABASE_ANON_KEY from Supabase Dashboard

# 3. Generate Supabase types from live schema
npm run types:gen

# 4. Run dev server
npm run dev
```

Open <http://localhost:3000>
